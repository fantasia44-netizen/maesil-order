# AutoTool + PackFlow 마스터 로드맵

> **작성일**: 2026-03-23 (v2 — 축산MES 후순위 반영)
> **범위**: AutoTool (ERP/WMS) + PackFlow (3PL SaaS)
> **현재 상태**: AutoTool 80% 운영 중 / PackFlow 60% 코드 완성
> **목표**: AutoTool 완성 → 3PL 완성 → SaaS 상용화
> **후순위**: 축산 MES (설계 완료, 코드 착수는 AutoTool+3PL 완성 후)

### 핵심 의존성 (순서 어기면 안 됨)
1. **AutoTool 주문처리 완성** → 3PL API 주문수집 이식 (코드 재사용)
2. **Phase 2A 실제원가** + **2C-1 간접비** → Phase 2C-2 경영P&L
3. **Phase 2A~2D 전체** → Phase 2E CEO 대시보드
4. **020 SQL 실행** → 과금 UI 개발 (DB 테이블 필수)
5. **SaaS 격리 완료** → 2사+ 운영 (격리 없이 입주 금지)

### 독립 병렬 가능
- Phase 2A ↔ Phase 2B (서로 무관)
- Phase 2B ↔ Phase 2C-1 (서로 무관)
- Phase 2D ↔ Phase 2A/2B/2C-1 (서로 무관)
- 3PL 과금 UI ↔ AutoTool Phase 2 (서로 무관)
- 3PL 현장화면 ↔ 3PL 과금 (서로 무관)
- SaaS RLS ↔ Phase 2 기능개발 (서로 무관)
